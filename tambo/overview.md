# Tambo AI Documentation Summary

## What is Tambo?

Tambo is a React package designed for constructing AI-powered applications featuring generative user interfaces where interaction occurs through natural language. It's suitable for building AI chats, copilots, or custom AI experiences.

## Core Components

**React Package (`@tambo-ai/react`)**: Provides hooks for managing components and tools available to Tambo, sending user messages to the API, and handling responses and conversation threads. Applications wrap their components with `<TamboProvider/>` and utilize hooks for messaging functionality.

**Tambo API**: Handles message processing, LLM interactions, conversation management, tool orchestration, and user authentication.

**UI Component Library**: Pre-built React components with integrated Tambo functionality for common AI interfaces like chat and forms, accessible at ui.tambo.co.

## Key Features

The platform offers:
- Dynamic React component rendering responding to user messages
- Real-time streaming for AI-generated content
- Automatic conversation storage
- AI-integrated state management
- User guidance through suggested actions
- Automatic tool orchestration during response generation
- Ready-to-use components installable via CLI
- Customizable model parameters and reasoning capabilities
